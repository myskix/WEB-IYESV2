## üè¢ Tentang Proyek
Website ini dibangun sebagai solusi *end-to-end* untuk kebutuhan digital IYES Foundation. Terbagi menjadi dua antarmuka utama:
1.  **Public Frontend:** Halaman landing page responsif untuk pengunjung umum, menampilkan profil, berita, dan pendaftaran event.
2.  **Admin Dashboard:** Panel kontrol berbasis **FilamentPHP** untuk administrator mengelola seluruh konten tanpa menyentuh kode.

## üöÄ Fitur Utama

### üåç Sisi Publik (Frontend)
- **Responsive Design:** Tampilan adaptif untuk Mobile, Tablet, dan Desktop.
- **Lazy Loading Optimization:** Optimasi LCP (*Largest Contentful Paint*) pada gambar dan slider untuk kecepatan akses.
- **Dynamic Content:** Berita, Program, dan Pengumuman yang terupdate real-time dari database.
- **Interactive UI:** Animasi halus dan transisi antar elemen menggunakan AlpineJS.

### üîê Sisi Admin (Backoffice)
- **Role-Based Access Control (RBAC):** Manajemen hak akses untuk Super Admin dan Editor Divisi.
- **Content Management System (CMS):** CRUD (Create, Read, Update, Delete) untuk Berita, Program, dan Galeri.
- **Media Library:** Manajemen upload gambar dan dokumen terpusat.
- **Reporting:** Export data pendaftar atau laporan kegiatan.
